# FLL UNEARTHED Mission Flashcards

An interactive training app for FLL Team 71319 Airborne Inventors to master the 2025-2026 UNEARTHED season missions.

## Features

### 📚 Interactive Flashcards
- Visual mission cards with flip animations
- Mission images from official game guide
- Detailed scoring breakdowns
- Strategy tips and attachment requirements
- Keyboard shortcuts for quick navigation

### 🎯 Quiz Modes
- **Mission Identification**: Identify missions from images
- **Points Challenge**: Match missions with point values
- **Strategy Quiz**: Test attachment types and priorities
- **Ultimate Challenge**: Mixed questions from all categories

### 📊 Progress Tracking
- Track cards studied and quiz performance
- Mission mastery percentages
- Achievement system
- Study time tracking
- Persistent progress saving

### 📖 Reference Guide
- Sortable mission grid (by number, points, difficulty, priority)
- Filterable by attachment type
- Competition summary with total points
- Quick access to any mission

## Keyboard Shortcuts

- `Space` or `F` - Flip card
- `←` or `A` - Previous mission
- `→` or `D` - Next mission
- `1-9` - Jump to mission
- `?` - Show shortcuts
- `Esc` - Close modal

## Deployment

### GitHub Pages Setup

1. Push this repository to GitHub
2. Go to Settings → Pages
3. Source: Deploy from a branch
4. Branch: main, folder: / (root)
5. Save and wait for deployment

The app will be available at: `https://[username].github.io/fll-mission-flashcards/`

## Local Development

Simply open `index.html` in a modern web browser. No build process required!
